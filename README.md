# fullScreenScroll
fullScreenScroll是多个平级ViewController横向来回滑动切换的组件。
####类：
##### CZSegmentViewController - 管理子类控制器与UIPageViewController对象
##### CZScrollView - 针对UIScrollView的封装，可以横向、竖向滑动
##### HMSegmentControl - 页面指示器
